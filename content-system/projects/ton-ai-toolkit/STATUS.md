# TON AI Toolkit — Content Kit

## Project Summary
Three-part TON blockchain hackathon submission: AI agents that actually transact on-chain. "The Stack Is Ready, The Apps Aren't."

## Key Facts
- **Repo**: https://github.com/mbaneshi-labs/ton-ai-toolkit
- **Submission Deadline**: March 25, 2026
- **Stack**: TypeScript, grammY, Claude API, PostgreSQL, Zod, Vitest
- **License**: MIT

## Three Submissions
1. **TON DAO Agent** — Telegram bot for group treasury management via natural language + on-chain multisig
2. **@ton-intent/core** — NL-to-blockchain transaction framework (<5KB gzipped)
3. **@ton-mcp/server** — Read+write MCP server for TON with 20 tools (the only one that exists)

## Unique Selling Points (use in content)
1. **Zero of top AI bots transact on TON on-chain** — this fills that gap
2. **Natural language to verified blockchain transactions** — say "send 5 TON to alice" and it builds, validates, signs
3. **MCP server** — Claude Desktop can interact with TON blockchain directly
4. **Multisig V2** — real DAO governance, not toy demos
5. **Three complementary submissions** — bot, framework, and infrastructure
